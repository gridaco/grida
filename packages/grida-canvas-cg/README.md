# @grida/cg

Core Graphics Typings and Utilities

## Installation

```bash
npm install @grida/pixel-grid
# or
yarn add @grida/pixel-grid
# or
pnpm add @grida/pixel-grid
```

## Usage

```ts
import cg from "@grida/cg";

const boxShadow: cg.BoxShadow = {
  color: { r: 0, g: 0, b: 0, a: 0.25 },
  offset: [0, 4],
  blur: 4,
  spread: 0,
};
```
