**Moved.** The context menu is one of two menu surfaces over the
command registry; both are now specified together in
[menu.md](./menu.md). The former `CTX-1`…`CTX-5` contracts are the
`MENU-*` contracts there (`CTX-1`→`MENU-1`, `CTX-2`→`MENU-2`,
`CTX-3`→`MENU-4`, `CTX-4`→`MENU-5`, `CTX-5`→`MENU-6`).

This stub is kept for one release and will be removed.
