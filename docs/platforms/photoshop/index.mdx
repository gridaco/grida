# Bridged Photoshop plugin for asset sync and Handoff

## When to use Photoshop?

- Creating origin graphic assets work well with Bridged and Photosop. Your asset can be synced and be managed on Bridged Cloud, can be imported, exported from other design platforms such as Figma and Sketch.

## What is not supported for Photoshop?

- Design to code. - only asset management is supported with Photoshop. Please use Figma, Sketch or XD for UI Design.
