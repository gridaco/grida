# Grida Assistant - Figma Plugin

Grida Assistant for Figma is a powerful plugin that helps you convert your Figma designs into production-ready code and provides essential design tools. It streamlines your workflow by offering code generation, icon management, and design validation features.

## Key Features

- **Figma to Code**: Convert your Figma designs into React components and Flutter widgets
- **Icon Library**: Access and use a comprehensive collection of icons with drag-and-drop functionality
- **Design Linter**: Validate your designs with automated checks and best practices

## Getting Started

1. Install the Grida Assistant plugin from the [Figma Community](https://www.figma.com/community/plugin/896445082033423994/assistant-by-grida)
2. Open your Figma project
3. Access Grida Assistant through the plugins menu
4. Choose your desired feature:
   - Generate code from your designs
   - Browse and use icons
   - Run design validation
